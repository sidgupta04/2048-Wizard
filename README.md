# 2048 Wizard

Created by: Siddharth Gupta, Ria Achutan, Puru Gajare

2048 Wizard explores different artificial intelligence approaches for solving the game 2048, a puzzle game played on a 4×4 grid where tiles with the same value merge to form larger tiles until the 2048 tile is reached. This project compares three strategies: Expectimax, a search-based planning algorithm; Deep Q-Network (DQN), a reinforcement learning model that learns from gameplay experience; and a hybrid approach where DQN is trained on Expectimax-generated gameplay before being fine-tuned with reinforcement learning. The goal is to analyze how planning-based and learning-based methods differ in performance, computational cost, and gameplay behavior.
